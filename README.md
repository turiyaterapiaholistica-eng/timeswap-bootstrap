# TimeSwap Bootstrap

Repositorio minimo para publicar o arquivo `timeswap-session.json`.

URLs fixas:

- bootstrap JSON: `https://raw.githubusercontent.com/turiyaterapiaholistica-eng/timeswap-bootstrap/main/timeswap-session.json`
- página humana estável: `https://turiyaterapiaholistica-eng.github.io/timeswap-bootstrap/`

Uso:

- `timeswap live` atualiza esse arquivo
- o APK ou app web fixo le esse JSON por uma URL publica estavel
- `index.html` lê o JSON e redireciona para a sessão pública atual

Arquivo principal:

- `timeswap-session.json`
- `index.html`

Observacao:

- para a URL `github.io` funcionar, ative GitHub Pages neste repositório usando `main` e a pasta `/ (root)`
